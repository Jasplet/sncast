SN-CAST documentation
=====================

SN-CAST (Seismic Network Capability Assessment Software Tool) is a Python library for modelling the earthquake detection capability of Seismic Networtk, Arrays and DAS.

The original version of this software is available `on Github here <https://github.com/moellhoff/sncast>`_. This version is described in `Molhoff et al., (2019) <https://doi.org/10.1007/s10950-019-09819-0>`_.
